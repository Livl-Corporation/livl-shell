This project is developed in C language, so you need to have a C compiler installed on your machine. It would be better to run it on a Linux distribution, but it is possible to run it on Windows with the W\+SL (Windows Subsystem for Linux).


\begin{DoxyItemize}
\item {\ttfamily gcc} \+: {\ttfamily sudo apt install gcc}
\item {\ttfamily make}\+: {\ttfamily sudo apt install make}
\end{DoxyItemize}

\begin{quote}
To install the W\+SL, follow this \href{https://docs.microsoft.com/en-us/windows/wsl/install-win10}{\texttt{ tutorial}}. \end{quote}
\hypertarget{index_autotoc_md2}{}\doxysection{🚀 Quick start}\label{index_autotoc_md2}

\begin{DoxyItemize}
\item In the root folder, go to the lab folder you want to use (ex\+: {\ttfamily cd 02-\/proc})
\item Run {\ttfamily make} to compile the project, it will create an executable file in the {\ttfamily bin} folder called {\ttfamily livl-\/shell}
\item Run {\ttfamily ./bin/livl-\/shell} or {\ttfamily make run} to run the executable
\item Run {\ttfamily make clean} to clean the project

\item {\ttfamily cd} \+: change the current directory
\item {\ttfamily pwd} \+: print the current directory
\item {\ttfamily cat $<$$<$E\+OF} \+: let the user enter text until he types {\ttfamily E\+OF} and display it (thanks to the cat command)
\item {\ttfamily echo} \+: display a line of text
\item {\ttfamily ls -\/l $>$ output.\+txt} \+: redirect the output of a command to a file
\item {\ttfamily pwd $>$$>$ output.\+txt} \+: redirect the output of a command to a file (append)
