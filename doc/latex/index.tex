This project is developed in C language, so you need to have a C compiler installed on your machine. It would be better to run it on a Linux distribution, but it is possible to run it on Windows with the W\+SL (Windows Subsystem for Linux).


\begin{DoxyItemize}
\item {\ttfamily gcc} \+: {\ttfamily sudo apt install gcc}
\item {\ttfamily make}\+: {\ttfamily sudo apt install make}
\end{DoxyItemize}

\begin{quote}
To install the W\+SL on Windows, follow this \href{https://docs.microsoft.com/en-us/windows/wsl/install-win10}{\texttt{ tutorial}}. \end{quote}
\hypertarget{index_autotoc_md2}{}\doxysection{🚀 Quick start}\label{index_autotoc_md2}
Follow these steps to quickly get started\+:


\begin{DoxyEnumerate}
\item {\bfseries{Compile the Project}}\+: Run the {\ttfamily make} command. This will compile the project and create an executable file called {\ttfamily livl-\/shell} in the {\ttfamily bin} folder.
\item {\bfseries{Run the Executable}}\+: You can run the executable by using the command {\ttfamily ./bin/livl-\/shell} or {\ttfamily make run}.
\end{DoxyEnumerate}

\begin{quote}
😀 {\bfseries{Insane feature}}\+: You can run the executable in batch mode by using the command {\ttfamily ./bin/livl-\/shell -\/c \char`\"{}ls -\/l\char`\"{}}. \end{quote}



\begin{DoxyEnumerate}
\item {\bfseries{Clean the Project}}\+: Use the {\ttfamily make clean} command to clean the project.
\end{DoxyEnumerate}

\begin{quote}
💡 {\bfseries{Tip}}\+: The command {\ttfamily make run} will compile and immediately run the executable. It is equivalent to running {\ttfamily make \&\& ./bin/livl-\/shell}. \end{quote}

\hypertarget{index_autotoc_md5}{}\doxysubsection{Basic Commands}\label{index_autotoc_md5}

\begin{DoxyItemize}
\item {\ttfamily ls}\+: Lists the files in the current directory.
\item {\ttfamily cd}\+: Changes the current directory.
\item {\ttfamily pwd}\+: Prints the current directory.
\item {\ttfamily echo}\+: Displays a line of text.
\end{DoxyItemize}\hypertarget{index_autotoc_md6}{}\doxysubsection{Input/\+Output Redirection}\label{index_autotoc_md6}

\begin{DoxyItemize}
\item {\ttfamily ls -\/l $>$ output.\+txt}\+: Redirects the output of a command to a file.
\item {\ttfamily pwd $>$$>$ output.\+txt}\+: Appends the output of a command to a file.
\item {\ttfamily wc -\/l $<$ output.\+txt}\+: Counts the number of lines in the file.
\end{DoxyItemize}\hypertarget{index_autotoc_md7}{}\doxysubsection{Pipelines}\label{index_autotoc_md7}

\begin{DoxyItemize}
\item `ls -\/l $\vert$ grep '.txt\textquotesingle{}\`{}\+: Redirects the output of a command to another command. You can use single, double, or no quotes.
\end{DoxyItemize}\hypertarget{index_autotoc_md8}{}\doxysubsection{Command Sequencing}\label{index_autotoc_md8}

\begin{DoxyItemize}
\item {\ttfamily sleep 4 \&\& echo \char`\"{}\+Second command executed\char`\"{}}\+: Executes a command after another one.
\item {\ttfamily false \&\& echo \char`\"{}\+This won\textquotesingle{}t be executed\char`\"{}}\+: Executes a command after another one only if the first one succeeded.
\item {\ttfamily false $\vert$$\vert$ echo \char`\"{}\+This will be executed\char`\"{}}\+: Executes a command after another one only if the first one failed.
\item {\ttfamily echo \char`\"{}\+Command 1\char`\"{}; sleep 3; echo \char`\"{}\+Command 2\char`\"{}; ls -\/l}\+: Executes multiple commands regardless of the success of the previous ones.
\end{DoxyItemize}\hypertarget{index_autotoc_md9}{}\doxysubsection{Batch Mode}\label{index_autotoc_md9}
\begin{quote}
⚠️ The command should be enclosed in quotes to ensure it is passed as a single argument \end{quote}



\begin{DoxyItemize}
\item {\ttfamily ./livl-\/shell -\/c \char`\"{}ls -\/l$\vert$grep txt \&\& echo heyyy\char`\"{}}\+: Executes a list of commands from the command line.
\item {\ttfamily ./livl-\/shell -\/-\/command \char`\"{}echo livl\char`\"{}}\+: Executes a list of commands from the command line.
\end{DoxyItemize}\hypertarget{index_autotoc_md10}{}\doxysubsection{Background Execution}\label{index_autotoc_md10}

\begin{DoxyItemize}
\item {\ttfamily sleep 3 \& echo hey}\+: Executes a command in the background (the shell will not wait for the command to finish) and it will show you the job id of the background process (ex\+: {\ttfamily \mbox{[}1\mbox{]} 1234}).
\begin{DoxyItemize}
\item {\ttfamily pwd}\+: Running this command will display the job id of the background process terminated (ex\+: {\ttfamily \mbox{[}1\mbox{]} done sleep 3}).
\end{DoxyItemize}
