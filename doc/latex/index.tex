\begin{DoxyItemize}

\item {\ttfamily cd}\+: Changes the current directory.
\item {\ttfamily pwd}\+: Prints the current directory.

\end{DoxyItemize}\hypertarget{index_autotoc_md7}{}\doxysubsubsection{Input/\+Output Redirection}\label{index_autotoc_md7}

\begin{DoxyItemize}
\item {\ttfamily ls -\/l $>$ output.\+txt}\+: Redirects the output of a command to a file.
\item {\ttfamily pwd $>$$>$ output.\+txt}\+: Appends the output of a command to a file.
\item {\ttfamily wc -\/l $<$ output.\+txt}\+: Counts the number of lines in the file.
\item {\ttfamily cat $<$$<$ E\+OF} \+: let the user enter text until he types \textquotesingle{}E\+OF\textquotesingle{} (You can change E\+OF by whatever you want). The text is then displayed on the screen thanks to the cat command.
\end{DoxyItemize}\hypertarget{index_autotoc_md8}{}\doxysubsubsection{Pipelines}\label{index_autotoc_md8}

\begin{DoxyItemize}
\item `ls -\/l $\vert$ grep '.txt\textquotesingle{}\`{}\+: Redirects the output of a command to another command. You can use single, double, or no quotes.
\end{DoxyItemize}\hypertarget{index_autotoc_md9}{}\doxysubsubsection{Command Sequencing}\label{index_autotoc_md9}

\begin{DoxyItemize}
\item {\ttfamily sleep 4 \&\& echo \char`\"{}\+Second command executed\char`\"{}}\+: Executes a command after another one.
\item {\ttfamily false \&\& echo \char`\"{}\+This won\textquotesingle{}t be executed\char`\"{}}\+: Executes a command after another one only if the first one succeeded.
\item {\ttfamily false $\vert$$\vert$ echo \char`\"{}\+This will be executed\char`\"{}}\+: Executes a command after another one only if the first one failed.
\item {\ttfamily echo \char`\"{}\+Command 1\char`\"{}; sleep 3; echo \char`\"{}\+Command 2\char`\"{}; ls -\/l}\+: Executes multiple commands regardless of the success of the previous ones. The {\ttfamily ls -\/l} won\textquotesingle{}t be executed because the shell is limited to run a maximum of 3 commands in a row.

Livl-\/shell provide a basic alias support. \mbox{\hyperlink{structAlias}{Alias}} can be defined only by editing the {\ttfamily aliases.\+txt} file. The syntax is the following\+:

\begin{quote}

For example, if you want to create an alias for the {\ttfamily ls -\/l} command, you can add the following line to the {\ttfamily alias.\+txt} file\+:


\begin{DoxyCode}{0}

\end{DoxyCode}


Then, you can run the {\ttfamily ll} command to execute the {\ttfamily ls -\/l} command.

\begin{quote}
You can input arguments with your alias, as well as use pipes and redirections. For example, if you input {\ttfamily ll -\/m} it will be interpreted as {\ttfamily ls -\/l -\/m}. \end{quote}


\begin{quote}
